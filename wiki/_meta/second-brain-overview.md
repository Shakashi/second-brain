# Second Brain Overview

Ce vault est un second brain maintenu par agents. Il est conçu pour fonctionner dans Obsidian et avec des agents type Morpheus, Claude Code ou Codex.

## Structure principale

- [[ingestion-workflow|Ingestion Workflow]] décrit comment intégrer les documents déposés.
- `raw/` conserve les sources originales normalisées.
- `wiki/` contient la connaissance synthétisée.
- `wiki/index.md` est la carte de navigation.
- `wiki/log.md` est le journal des changements.

## Usage attendu

1. L'utilisateur dépose des sources dans `inbox/`.
2. Un agent prépare ces sources vers `raw/`.
3. L'agent extrait les informations utiles.
4. L'agent met à jour les pages existantes ou crée de nouvelles pages.
5. Les liens Obsidian rendent la connaissance navigable.
